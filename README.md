# 📦 Supply Chain Optimizer

Инструмент для оптимизации цепочки поставок между складами и магазинами.

## Что делает

- Находит оптимальный план поставок с минимальной стоимостью
- Учитывает запасы складов и спрос магазинов
- Визуализирует граф поставок
- Читает данные из CSV файла

## Как работает

Задача решается методом **линейного программирования**:
- Переменные — количество товара на каждом маршруте
- Цель — минимизировать общую стоимость доставки
- Ограничения — не превышать запасы складов и удовлетворить спрос магазинов

## Стек

- Python 3.12
- PuLP — линейное программирование
- pandas — работа с данными
- networkx — визуализация графа
- matplotlib — отрисовка

## Установка
```bash
pip install pulp pandas networkx matplotlib
```

## Запуск
```bash
python model.py --data data/example.csv
```

## Формат данных

CSV файл со столбцами:
- `from` — название склада
- `to` — название магазина  
- `supply` — запас склада
- `demand` — спрос магазина
- `cost` — стоимость доставки
